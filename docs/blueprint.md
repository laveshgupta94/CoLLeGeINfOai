# **App Name**: CampusConnect AI

## Core Features:

- AI Chatbot: An AI-powered chatbot that answers student queries regarding events, exams, attendance, and general college information. This chatbot will act as a tool using admin provided documents to accurately respond.
- Multi-language Support: Automatic language detection and response in multiple languages including English, Hindi, Marathi, Bengali, Tamil, and Telugu.
- Student Dashboard: A dashboard providing an overview of upcoming events, exam schedules, attendance status, and quick access to the chatbot.
- Admin Dashboard: A secure admin dashboard for managing events, exam timetables, attendance rules, and the AI knowledge base.
- Event and Exam Management: Admin tools to add, update, and delete events and exam timetables, ensuring students are always informed.
- Attendance Prediction: A tool for students to predict their attendance status and see how many more classes they need to attend to meet the required percentage.
- Data Storage: Store student questions, admin uploaded data, student events, exam data, and student attendance in Supabase.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to promote trust and learning.
- Background color: Light gray (#F0F4F7), nearly desaturated blue, to ensure readability and a clean look.
- Accent color: A vibrant orange (#FFAB40), analogous to yellow, to highlight important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif font for a blend of modernity and approachability, for both headings and body text.
- Code font: 'Source Code Pro' for displaying any code snippets related to technical instructions.
- Simple, clear icons to represent different features and data points in the student dashboard.
- Clean, card-based layouts for easy access to information on the student dashboard.
